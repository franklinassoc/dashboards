Project Guidance

  1. EBRRA staff access the "New Request Form" page of this website to enter each new case; click "Submit" .
  2. A confirmation email is sent to staff person submitting the request and to Franklin Associates, alerting them to the request.
  3. System creates a case folder within Google Drive.
  4. System generates a summary report of the request data and sends it to Google Drive. It is placed in the corresponding case folder.
  5. EBRRA staff access this Google Drive case folder and upload supporting documents for each property, as required.
  6. Franklin staff examine requests for completeness; requesting supplemental information as required.
  7. When complete, Franklin staff forward a notification to roof inspectors. Case is queued for initial inspection scheduling.
  8. Inspection findings are summarized in a pair of reports: one details the scope with itemized costs, the second lists the scope only.
  9. Inspectors deliver reports to Franklin Associates via email.
  10. Accepted roof inspection reports are moved to the corresponding project case folder and a notice is sent to EBR Redevelopment Authority of the status.
  11. A decision is made whether the case qualifies for roof replacement or repair. ($___K cap?)
  12. If "yes", the case enters the roofing installation management process below. If "no", the applicant is notified of the decision by EBR Redevelopment Authority and case files are archived.

(rough draft)
  1. Bid review and award. Contractor selection.
    • Contract signing.
    • Scheduling of work.
    • Notice to Proceed
    • Permitting
  2. Work Initiation
    • Homeowner notification
    • Property protection
    • Remove old shingles & materials
  3. Interim Inspection
    • Inspect roof decking & structure (contractor)
    • Photo / video documentation of condition of roof decking. (deliver to Franklin)
    • Minor carpentry repairs as required. (Photo document.)
    • Change orders for major carpentry repairs (contact Franklin)
  4. Install New Roof
    • Prep surface for new roofing materials, sweep off debris
    • Collect and properly dispose of debris, old nails, etc.
    • Install drip edge
    • Install felt / underlayment
    • Flashing, valleys, sealing
    • Install new shingles
    • Sealing as required
    • Install vents. Reset turbines, antenna, TV dish, etc. as required.
    • Site clean up
  5. Project close-out
    • Final Inspection
    • Punch List
    • Payments
